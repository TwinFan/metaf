ReportType
==========

.. cpp:namespace-push:: metaf

.. cpp:enum-class:: ReportType

		Autodetected METAR or TAF report type.

		.. cpp:enumerator:: UNKNOWN

			Unable to detect a report type (e.g. due to malformed report).

		.. cpp:enumerator:: METAR

			Report is METAR.

		.. cpp:enumerator:: TAF

			Report is TAF.
